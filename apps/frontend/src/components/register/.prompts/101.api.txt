아래의 조건을 모두 적용하여, 아래의 요구사항을 모두 구현할 것.
구현 결과를 체크리스트로 반환할 것.

==============================================

조건-커서룰) 아래의 커서룰을 적용하여 작업하고, 이 작업이 끝나면 해당 rules 적용 결과를 체크리스트로 반환할 것.
            - .cursor/rules/01-common.mdc

==============================================


조건-파일경로) 구현될 TSX 파일경로: src/components/register/register.tsx
조건-파일경로) Provider 파일경로: src/commons/providers/auth-context/auth-context.tsx

조건-API) 회원가입 엔드포인트 경로: POST /api/auth/register

핵심요구사항) 회원가입 페이지에 목데이터들을 제거하고, 회원가입 엔드포인트를 호출하여, 회원가입 기능을 구현할 것.
            - 회원가입 성공 시, authContext에 저장된 user 정보를 초기화하고, /dashboard 페이지로 이동할 것.
            - 회원가입 성공 시, 회원가입 정보를 localStorage에 저장할 것.
              - key: accessToken
              - value: 회원가입 엔드포인트 응답에서 가져온 accessToken
              - key: refreshToken
              - value: 회원가입 엔드포인트 응답에서 가져온 refreshToken

핵심요구사항) authContext.tsx 의 register 함수의 목데이터를 제거하고, 현재 프롬프트에 맞게 수정할 것.
